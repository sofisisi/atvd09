O que é Formulário HTML?✧˖*°࿐

O formulário HTML é um formulário de preenchimento de dados ou que resulta em uma ação desejada utilizando a linguagem de marcação HTML. É formado por um ou mais widgets que são campos de textos, caixas de seleção, botões, radio buttons e checkboxes utilizando ferramentas do próprio HTML. Dessa forma, o usuário pode interagir com a página ao executar ações através desses formulários.

Alguns exemplos de formulários podem ser as caixas de texto, botões de busca em sites e muito mais.

Como inserir um Formulário HTML?✧˖*°࿐

A tag FORM é a abertuda do formulário HTMl e a tag /FORM é o fechamento. Podem ser atruibuidos a essa tag o atributo METHOD e o ACTION.

ACTION define o local (por uma URL) onde serão enviados os dados recolhidos do formulário.

METHOD define o método HTTP que o formulário HTML irá lidar com os dados que recebeu. Podem ser dois tipos, GET e POST.

O método GET é usado pelo navegador para solicitar que o servidor envie de volta um determinado recurso.Nesse caso, o navegador envia um corpo vazio. Portanto, já que o corpo fica vazio, se um formulário for enviado através do método GET, os dados serão reconhecidos pelo servidor através da URL.
Dessa forma, podemos perceber que ao enviar um formulário com os campos Nome e Idade, a URL iria se parecer com algo como: url?nome=valor&idade=value.

O método POST é utilizado no navegador para conversar com o servidor. Os dados são enviados ao servidor através do corpo da solicitação HTTP. Também é realizada uma solicitação de resposta.
Esse método não inclui o corpo na URL. Portanto, os dados enviados não ficarão visíveis na URL.

Input✧˖*°࿐

A tag INPUT em HTML representa os campos de entrada de dados de um formulário.

O atributo mais importante é o atributos type. Esse atributo representa o tipo de campo de entrada ao qual essa tag pertence e como ele se comporta.
Outro atributo importante para a tag é o atributo name, que define um nome para o campo de entrada.
Outro atributo que também merece destaque é o placeholder, pois esse atributo permite digitarmos uma “descrição” que será visualizada enquanto o campo de entrada não possuir nenhum valor.

Tipos de campos INPUT✧˖*°࿐

Type button define um botão

Type checkbox define uma caixa de checagem oie! Tudo bem com você?

Type color define uma caiza de cores

Type date define um campo de data

Type email define um campo de email

Type file define um campo para upload de arquivos

Type password define um campo de senha

Type text define um campo de texto
